Bot Token Setup (Recommended):

2. Create a new app or select existing
3. Go to "OAuth & Permissions"
4. Add scopes: chat:write, chat:write.public
5. Install to workspace and copy the Bot User OAuth Token
6. Invite the bot to your channel: /invite @YourBotName
